Инструкция по синхронизации /var/www/html

Предпосылки: rsync должна быть установлена на всех машинах.

  • @local ~]$ rsync -a -e "ssh -p 7967" html/ user@host:html/
  • на сервере синхронизируем каталоги: remote ~]$ sudo rsync -a html/ /var/www/html/

Вода: Сколько это занимает времени? С того момена как вы внесли измениния у себя в студии, до момента публикации на сервере, проходит не больше 20 секунд. Все очень быстро. Можно назвать это реал-тайм.

Команда scp

scp - копирует файлы между хостами в Сети. Работает подобно команде cp

Примеры:

  • Файл с удаленного хоса к себе в текущию дерикторию - ~]$ scp user@host:path_file path_copy_past
  • Каталог c удаленного хоста через другой порт - ~]$ scp -r -P 2871 user@host:path_file path_copy_past
Внимание: для scp порт указываеться с большой P
