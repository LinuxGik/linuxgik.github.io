Конвертация png в svg

~]$ dnf install potrace

]$ convert stiven_Born.jpg stive_born.pnm

]$ potrace stive_born.pnm -s -o stive_born.svg

Готова svg но без цвета. Чтобы сохраить цвет ипортируем image.pnm в Inkscape и сохраняем в.svg

В Inkscape нужно подгнать размер страницы под изображение. Заходим в меню Файл-Свойства документа. Там нажимаем кнопку "Resize to content"

Подсказки

teg=css общее. (*) - универсальный селектор, предназначен для всех элемнтов на странице. 


								:root {
									box-sizing: border-box; /* --- применение к коренвому элементу.
								}

								*,
								::before,
								::after {
									box-sizing: inherit;
								}

box-sizing по умолчанию content-box. content-box - размер контента. border-box - рамер всего бокса (контента, отступов и границ).

О дизайне веб-сайта

ВВЕДЕНИЕ

Люди не любят ждать когда страница долго грузиться.

Для большинства Сеть это библиотека а не место для развлечений.

Акцентироваться на информации а не на графике

Дизайн должен быть простой, незаграмажденный, стандартизирован и помогать найти нужную информацию.

Компановка страницы должна облегчать чтение.

Без вычурной графики и мультимедиа.

ВОПРОС — ОТВЕТ

Что такое дизайн веб-сайта?

В первую очередь это дизайн ниформации. Это такая организация контента, чтобы можно было легко найти и прочитать нужные сведения.

Что в основном, делают люди в Сети?

Ищут информацию а потом читают ее.

В дизайне веб-сайта нужно избегать чрезмерно изощренных цветовых схем, мелких шрифтов, непродуманную компановку страницы.

ЧТО НУЖНО ДЕЛАТЬ

Дизайн с ориентацией на читателя

Ориентироваться нужно на запросы посетителей, на то зачем люди пришли на ваш сайта. Помните, человек который зашел на ваш сайт:

  • хочет найти какую-то информацию;
  • будет читать по диагонали, быстро пермещаться с одного фрагмента текста на другой;
  • обычно огрнаничен во времени и ему непонравиться долго ждать загрузку страницы;
  • иногда настроен скептически из-за своего предыдущего отрицательного опыта, ожидая увидеть массу безполезного материала.

Подкючение шрифта

@font-face { font-family: "myFont"; src: url("myFont.woff"); } - указывает шрифты для загрузки. Размещаем его в начале css-кода.

Теперь его можно использовать. html {font-family: "myFont", "Bitstream Vera Serif", serif;}

Так можно подключать несколько шрифтов, но для каждого надо создвать свой @font-face
