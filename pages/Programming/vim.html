Командный режим

Основное

:w new_name/code> - записать файл под новым именем

> :r name.txt - прочитать содержимое файла name.txt и вставить его в редактируемый текст. > :e file_name.txt - начать редактирование еще одного файла. > :ls - показать список редактируемых файлов (активных буферов) > :b - перейти к буферу номер N Ctrl + ^ - быстрое переключение между двумя активными буферами прим: при этом, по умалчанию редактор настроен так, чтобы текущий файл был сохранен. Это можно отменить командой :set hidden :set ---- laststaus=1 - уберет строку состояния. 2 - вернет строку состояни. Многооконность -------------- :split - cоздать новое окно (^Ws) :vslit - по вертикали (^Wv) прим: можно использовать с именами файлов, тогда будет открывать новые окна с новыйм файлом. Перемещение по окнам: ^W(hjkl)
