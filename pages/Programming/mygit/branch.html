Ветки Git

Имена веток

Ветка по дефолту в репе называется main, Git создаёт ее при инициализации репозитория. Большинстов разрабов тут хранит самую устойчивую и надежную линию разработки.

Проверить подходит ли имя которым вы хотите назвать ветку git check-ref-format --branch ≶branchname-shorthand>

Использование веток

Веток может быть много, но активная может быть только одна.

Создание веток

Новая ветка основывается на существующей фиксации в репозитории.

Базовая форма команды создания ветки.

git branch имя_новой_ветки [начальная фиксация]

Примечание: если начальная фиксация не указана, используется самая последняя фиксация.

Вывести имена веток которые есть в репорзитории

git brach

Переключится на другую ветку

git checkout имя
