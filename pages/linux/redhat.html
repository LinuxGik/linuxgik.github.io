Создаем запись в базе terminfo на примере терминала alacritty

			
				user@local ~]$ infocmp > alacritty.terminfo  # export Alacritty's Terminfo
				user@local ~]$ scp alacritty.terminfo user@remote-host:~/  # or any other method to copy to the remote host
				
				user@remote ~]$ tic -x alacritty.terminfo  # import Terminfo for current user
				user@remote ~]$ rm alacritty.terminfo  # optional: remove Terminfo file

Ввод и вывод с помощью оболочки

stdin- стандартный ввод stdou - стандартный вывод stderr - стадартаня ошибка

">" ~]$ ls > file отправит вывод в file. Создаст file, если file уже есть затрет данные в нем.

">>"вывод в файл без затирания

~]$ set -C включить параметр оболочки предотварщающий затирание (+С отключить)

"|" отправить стандартный вывод на стандартый вход другой комаде $ head /proc/cpuinfo | tr a-z A-Z

"2> стандартная ошибка $ ls /fffffffff > f 2> e

"2>& направит ошибку туда же куда stdin ~]$ ls /fffffffff > f 2>&1
