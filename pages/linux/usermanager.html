Права доступа

Режим файла

Таблица обозначение прав доступа
Обозначение Описание для файла Описание для каталога
r файл доступен для чтение каталоге доступен для просмотра содержимого
w файл доспупен для записи каталог доступен для создания и удаления фалов
x исполняемый файл (можно запустить его как программу) позволяет обращаться к файлам внутри каталога
— ничего не обозначает

Вывести права доступа в цифрах:

~]$ stat -c %a /media/sf_Shared/
777

Смена владельца файла. Команда chown

Изменить владельца файла/каталога может только суперпользователь root

~]$ ls -l file_name.txt
~]$ -rw-r--r--. 1 marina marina 144 ноя  7 21:49 file_name.txt
~]$ sudo chown bob file_name.txt
~]$ -rw-r--r--. 1 bob marina 144 ноя  7 21:49 file_name.txt

Изменить группу файла/каталога. Команда chgrp

Изменить группу может владелец файла, но только на ту к которой он сам принадлежит.

marina@fedora~]$ ls -l file_name.txt
marina@fedora~]$ -rw-r--r--. 1 marina wheel 144 ноя  7 21:49 file_name.txt
marina@fedora~]$ groups
marina@fedora~]$ marina
marina@fedora~]$ chgrp marina file_name.txt
marina@fedora~]$ -rw-r--r--. 1 marina marina 144 ноя  7 21:49 file_name.txt
