Подключение домена к сайту на GitHub

Прежде чем настраивать личный домен с помощью поставщика DNS, добавьте личный домен на сайт GitHub Pages.
Настройка личного домена с помощью поставщика DNS без добавления личного домена в GitHub может привести к тому, что другой пользователь сможет разместить сайт на одном из поддоменов.

Кстати. А что такое поддомен?

Чтобы настроить www или личный поддомен, например www.example.com, необходимо добавить домен в параметры репозитория. После этого с помощью поставщика DNS настройте запись CNAME.

перейти на оф.документацию по настройки.
